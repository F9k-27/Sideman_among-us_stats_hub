# Sideman Among Us Stats Hub

[Sideman Among Us Stats Hub preview](Images/logo.png)

## About Sideman Among Us Stats Hub

This website tracks **Among Us statistics for the Sidemen**, including:

- Overall win rates  
- Player performance  
- Imposter combinations  
- Kill statistics  
- Player-vs-player interactions  

Explore detailed stats for each player and discover the most successful imposters and strategies.

👉 **[Open the application](https://f9k-27.github.io/Sideman_among-us_stats_hub/)**

---

## Features

- 📊 Overall game statistics dashboard  
- 👥 Player roster with individual profiles  
- 🔪 Favorite victims & biggest nemeses  
- 👽 Role-specific stats (Crewmate / Imposter / All Roles)  
- 🤝 Top imposter combinations  
- ⚡ Fast, clean interface
